Install instructions

Required Python packages:
av
pygame
pillow

Program installation:
1. Install Python.
2. Install the three required packages.
3. Start the program **"AI Video Last Frame Extractor v1.0.pyw"**.
4. Drag a video file into the program window and it will automatically extract the last frame from the video. The image will be saved next to the program.

The program is free. Anyone can use it, modify it, sell it, put it on a pizza, etc...
